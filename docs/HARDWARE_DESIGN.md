## 2. 传感器模块

### 2.1 电池温度传感器 ⭐ P0 安全关键

**型号：** NTC 热敏电阻 10kΩ (B3950)

**电路原理图：**
```
         3.3V
          │
         ┌┴┐
         │ │ R1 10kΩ 上拉
         └┬┘
          │
          ├───┬─── GPIO3 (NTC_Temp)
          │   │
         ┌┴┐ ┌┴┐ C1 100nF
         │ │ │ │ 滤波电容
         └┬┘ └┬┘
          │   │
          │  GND
          │
         ┌┴┐
         │ │ TH1 NTC 10kΩ B3950
         └┬┘
          │
         GND
```

**参数：**
| 参数 | 值 |
|------|-----|
| 标称阻值 | 10kΩ @ 25°C |
| B 值 | 3950K |
| 精度 | ±1% |
| 工作温度 | -40~125°C |

**安装位置：**
- 紧贴电池组中间（单节电池表面）
- 使用导热胶固定
- 远离发热元件（MOSFET、稳压器）

**温度计算：**
```cpp
float read_battery_temp() {
    int adc = analogRead(GPIO3);
    float resistance = 10000.0 * (4095.0 / adc - 1);
    float steinhart = 1 / (0.003354016 + 0.0003354016 * log(resistance));
    return steinhart - 273.15;  // 转换为°C
}
```

**保护阈值：**
| 温度 | 动作 |
|------|------|
| ≥60°C | 立即切断充电和负载 |
| ≥50°C | 降低充电电流，发送警告 |
| ≤0°C | 禁止充电 |
| ≤-10°C | 切断负载 |

**⚠️ 安全关键：** 此传感器为 P0 级安全配置，不可省略！

### 2.3 水位检测（简化版）⭐

**方案 A：浮球开关（推荐）**

**适用：** 可乐瓶/水箱

**原理：**
```
      可乐瓶
    ┌─────────┐
    │         │
    │    🎈   │ ← 浮球
    │    │    │
    │    └────┼──→ 磁铁
    │         │
    │      [干簧管] ← 瓶外固定
    │         │
    └─────────┘
    
水位下降 → 浮球下降 → 磁铁远离 → 干簧管断开 → 输出低电平
水位上升 → 浮球上升 → 磁铁靠近 → 干簧管吸合 → 输出高电平
```

**元件：**
| 元件 | 型号 | 数量 | 成本 |
|------|------|------|------|
| 浮球开关 | 常开型 | 1 | ¥5 |
| 磁铁 | 小圆形 | 1 | ¥1 |
| 上拉电阻 | 10kΩ | 1 | ¥0.05 |

**电路：**
```
3.3V ────┬──── 10kΩ 上拉
         │
         ├─── GPIO2 (Water_Level)
         │
        浮球开关
         │
        GND
```

**安装：**
1. 可乐瓶侧面开小孔
2. 浮球开关固定在瓶外

**原理：**
```
    可乐瓶
  ┌─────────┐
  │         │
  │    ┌┴┐  │ ← 探针 1（长）
  │    │ │  │
  │    └┬┘  │
  │     │   │
  │    ┌┴┐  │ ← 探针 2（短）
  │    │ │  │
  │    └┬┘  │
  │     │   │
  └─────┴───┘
        │
       GND
```

**元件：**
- 铜线/不锈钢线 2 根
- 10kΩ 上拉电阻

**电路：**
```
3.3V ────┬──── 10kΩ 上拉
         │
         ├─── GPIO2
         │
        探针 1（长）
         │
        水（导电）
         │
        探针 2（短）
         │
        GND
```

**优点：**
- 成本极低（¥0）
- 结构简单
- 适合可乐瓶

**缺点：**
- 探针可能腐蚀
- 需要定期清洁

**制作：**
1. 找两根铜线（或回形针）
2. 从瓶盖插入，一长一短
3. 长的接近瓶底，短的在 desired 水位
4. 用热熔胶固定并绝缘

### 3.2 充电管理（TP4056）⭐

**型号：** TP4056 线性锂电池充电 IC

**电路：**
```
USB-C 5V ──→┬── TP4056 IN+
            │
           ┌┴┐
           │ │ D1 TVS SMAJ5.0A
           └┬┘
            │
           GND

TP4056 接线：
IN+  ──→ USB 5V
IN-  ──→ GND
OUT+ ──→ 电池+
OUT- ──→ 电池-（通过保护板）
PROG ──→ 1.2kΩ ──→ GND（设置充电电流 1A）
```

**参数：**
| 参数 | 值 |
|------|-----|
| 输入电压 | 4.5-8V |
| 充电电压 | 4.2V ±1% |
| 充电电流 | 可调（默认 1A） |
| 封装 | SOP-8/DFN-8 |

**充电电流设置：**
```
R_PROG = 1200Ω → I_CHG = 1A
R_PROG = 2400Ω → I_CHG = 500mA
R_PROG = 600Ω  → I_CHG = 2A

公式：I_CHG = 1200 / R_PROG
```

**推荐模块：**
- TP4056 模块（带保护板）- ¥3-5
- 含充电/保护功能

### 4.2 元件选型计算 ⭐

#### MOSFET 选择

**520 水泵参数：**
- 工作电压：5V DC
- 工作电流：300mA
- 启动电流：约 500mA（瞬间）

**MOSFET 要求：**
```
Vds > 5V × 2 = 10V  （2 倍余量）
Id  > 300mA × 3 = 0.9A（3 倍余量）
Rds(on) < 0.1Ω（降低功耗）
Vgs(th) < 3.3V（ESP32 驱动）
```

**AO4606 参数：**
| 参数 | 值 | 说明 |
|------|-----|------|
| Vds | 30V | 漏源电压 >10V ✅ |
| Id | 6.9A | 漏极电流 >0.9A ✅ |
| Rds(on) | 0.028Ω | 导通电阻 <0.1Ω ✅ |
| Vgs(th) | 1.5-2.5V | 开启电压 <3.3V ✅ |
| Pd | 2W | 功耗余量充足 ✅ |

**功耗计算：**
```
P = I² × Rds(on)
P = (0.3A)² × 0.028Ω = 0.0025W

温升：可忽略不计（无需散热片）
```

**推荐：** AO4606（性能最好，成本适中）

---

#### 栅极电阻计算

**作用：**
1. 限制栅极充电电流
2. 防止 MOSFET 开关振荡
3. 保护 ESP32 GPIO

**计算：**
```
GPIO 输出电流限制：ESP32 最大 40mA

Rg = Vgpio / Ig
Rg = 3.3V / 0.02A = 165Ω

选择：100Ω（标准值，留余量）
```

---

#### 续流二极管选择

**作用：** 吸收水泵断电时的反电动势

**推荐：** 1N4007（便宜、可靠）或 SS14（肖特基，更快）

**安装方向：**
```
阴极（有横线）→ 接电源+
阳极（无横线）→ 接 MOSFET 漏极

即：阴极接水泵+，阳极接水泵-
```

### 4.4 功耗和续航

**系统功耗：**

**待机电流（Deep Sleep）：**
- ESP32: 10μA
- 传感器：0μA（断电）
- 总计：10μA

**工作电流：**
- ESP32 WiFi: 200mA
- 传感器：20mA
- 水泵：300mA
- 总计：520mA

**电池续航（单节 18650 2500mAh）：**
```
假设每天浇水 2 次，每次 30 秒：
- 工作时间：2 × 30 秒 = 1 分钟/天
- 工作电量：520mA × 1/60 小时 = 8.7mAh/天
- 待机电量：10μA × 24 小时 = 0.24mAh/天

总计：8.94mAh/天

续航：2500mAh ÷ 8.94mAh/天 ≈ 280 天 ⭐
```

## 5. 安全设计要点

### 5.1 温度监控

- NTC 紧贴电池表面
- 使用导热胶固定
- 远离发热元件
- 走线包地保护

### 5.2 过流保护

- 保险丝靠近电池正极
- 走线加粗（≥1.0mm）
- 便于更换（如使用插座）

### 5.3 浪涌保护

- TVS 靠近 USB 入口
- 到 GND 走线短而粗
- 5V 主电源线加粗

### 5.4 防水设计

- PCB 喷涂三防漆
- 外部接口使用防水接头
- 外壳接缝打密封胶
