## 3. 传感器模块

### 3.1 电池温度传感器 ⭐ P0 安全关键

**型号：** NTC 热敏电阻 10kΩ (B3950)

**电路原理图：**
```
         3.3V
          │
         ┌┴┐
         │ │ R1 10kΩ 上拉
         └┬┘
          │
          ├───┬─── GPIO3 (NTC_Temp)
          │   │
         ┌┴┐ ┌┴┐ C1 100nF
         │ │ │ │ 滤波电容
         └┬┘ └┬┘
          │   │
          │  GND
          │
         ┌┴┐
         │ │ TH1 NTC 10kΩ B3950
         └┬┘
          │
         GND
```

**参数：**
| 参数 | 值 |
|------|-----|
| 标称阻值 | 10kΩ @ 25°C |
| B 值 | 3950K |
| 精度 | ±1% |
| 工作温度 | -40~125°C |

**安装位置：**
- 紧贴电池组中间（两节电池之间）
- 使用导热胶固定
- 远离发热元件（MOSFET、稳压器）

**温度计算：**
```cpp
float read_battery_temp() {
    int adc = analogRead(GPIO3);
    float resistance = 10000.0 * (4095.0 / adc - 1);
    float steinhart = 1 / (0.003354016 + 0.0003354016 * log(resistance));
    return steinhart - 273.15;  // 转换为°C
}
```

**保护阈值：**
| 温度 | 动作 |
|------|------|
| ≥60°C | 立即切断充电和负载，蜂鸣器报警 |
| ≥50°C | 降低充电电流，发送警告通知 |
| ≤0°C | 禁止充电（低温充电会损坏电池） |
| ≤-10°C | 切断负载（低温放电容量极低） |

**⚠️ 安全关键：** 此传感器为 P0 级安全配置，不可省略！

**方案 A：浮球开关（推荐）**

**适用：** 可乐瓶/水箱

**原理：**
```
      可乐瓶
    ┌─────────┐

**电路：**
```
3.3V ────┬──── 10kΩ 上拉

         ├─── GPIO2 (Water_Level)

         │

### 4.2 充电管理（TP4056）⭐

**型号：** TP4056 线性锂电池充电 IC

**电路：**
```
USB-C 5V ──→┬── TP4056 IN+
            │
           ┌┴┐
           │ │ D1 TVS SMAJ5.0A
           └┬┘
            │
           GND

TP4056 接线：
IN+  ──→ USB 5V
IN-  ──→ GND
OUT+ ──→ 电池+
OUT- ──→ 电池-（通过保护板）
PROG ──→ 1.2kΩ ──→ GND（设置充电电流 1A）
```

**参数：**
| 参数 | 值 |
|------|-----|
| 输入电压 | 4.5-8V |
| 充电电压 | 4.2V ±1% |
| 充电电流 | 可调（默认 1A） |
| 封装 | SOP-8/DFN-8 |

**充电电流设置：**
```
R_PROG = 1200Ω → I_CHG = 1A
R_PROG = 2400Ω → I_CHG = 500mA
R_PROG = 600Ω  → I_CHG = 2A

公式：I_CHG = 1200 / R_PROG
```

**推荐模块：**
- TP4056 模块（带保护板）- ¥3-5
- 含充电/保护功能

### 5.2 元件选型计算 ⭐

#### MOSFET 选择

**520 水泵参数：**
- 工作电压：5V DC
- 工作电流：300mA
- 启动电流：约 500mA（瞬间）

**MOSFET 要求：**
```
Vds > 5V × 2 = 10V  （2 倍余量）
Id  > 300mA × 3 = 0.9A（3 倍余量）
Rds(on) < 0.1Ω（降低功耗）
Vgs(th) < 3.3V（ESP32 驱动）
```

**AO4606 参数：**
| 参数 | 值 | 说明 |
|------|-----|------|
| Vds | 30V | 漏源电压 >10V ✅ |
| Id | 6.9A | 漏极电流 >0.9A ✅ |
| Rds(on) | 0.028Ω | 导通电阻 <0.1Ω ✅ |
| Vgs(th) | 1.5-2.5V | 开启电压 <3.3V ✅ |
| Pd | 2W | 功耗余量充足 ✅ |

**功耗计算：**
```
P = I² × Rds(on)
P = (0.3A)² × 0.028Ω = 0.0025W

温升：可忽略不计（无需散热片）
```

**替代型号：**
| 型号 | Vds | Id | Rds(on) | 封装 | 立创编号 |
|------|-----|-----|---------|------|---------|
| AO4606 | 30V | 6.9A | 0.028Ω | SOIC-8 | C198134 |
| SI2302 | 20V | 2.5A | 0.05Ω | SOT-23 | C130766 |
| BSS138 | 50V | 200mA | 3.5Ω | SOT-23 | C198120 |
| IRLML2502 | 20V | 4.2A | 0.045Ω | SOT-23 | C130768 |

**推荐：** AO4606（性能最好，成本适中）

#### 栅极电阻计算

**作用：**
1. 限制栅极充电电流
2. 防止 MOSFET 开关振荡
3. 保护 ESP32 GPIO

**计算：**
```
GPIO 输出电流限制：ESP32 最大 40mA

Rg = Vgpio / Ig
Rg = 3.3V / 0.02A = 165Ω

选择：100Ω（标准值，留余量）
```

**功耗：**
```
Pg = V² / R = 3.3² / 100 = 0.11W

选择：0805 封装（0.25W 额定功率）✅
```

#### 续流二极管选择

**作用：** 吸收水泵断电时的反电动势

**水泵反电动势：**
```
水泵是感性负载
断电时会产生高压尖峰：V_spike = L × di/dt
可能达到 20-50V

需要二极管续流：
I_f > 水泵电流 = 300mA
V_rrm > 电源电压 = 5V
```

**1N4148 参数：**
| 参数 | 值 | 说明 |
|------|-----|------|
| I_f | 200mA | 正向电流 ⚠️ 偏小 |
| V_rrm | 100V | 反向电压 >5V ✅ |
| V_f | 1V | 正向压降 |

**更好的选择：**
| 型号 | I_f | V_rrm | 说明 |
|------|-----|-------|------|
| 1N4148 | 200mA | 100V | 勉强可用 |
| **1N4007** | **1A** | **1000V** | **推荐** ⭐ |
| SS14 | 1A | 40V | 肖特基（更快） |
| SS34 | 3A | 40V | 肖特基（大电流） |

**推荐：** 1N4007（便宜、可靠）或 SS14（肖特基，更快）

**安装方向：**
```
阴极（有横线）→ 接电源+
阳极（无横线）→ 接 MOSFET 漏极

即：阴极接水泵+，阳极接水泵-
```

### 5.4 常见问题

**Q1: MOSFET 发热严重？**

**A:** 
- 检查 Rds(on) 是否太大
- 检查栅极电压是否足够（应>3V）
- 检查续流二极管方向

**Q2: 水泵不工作？**

**A:**
- 检查 GPIO 是否有 3.3V 输出
- 检查 MOSFET 是否损坏
- 检查续流二极管是否短路
- 检查水泵是否损坏

**Q3: ESP32 重启？**

**A:**
- 续流二极管方向反了
- 水泵反电动势干扰
- 电源功率不足
- 增加输入电容（100μF）

**Q4: MOSFET 误导通？**

**A:**
- 栅极悬空（应加下拉电阻 10kΩ）
- 栅极走线太长（耦合干扰）
- 续流二极管失效

### 5.5 测试方法

**万用表测试：**

1. **断电测试**
   ```
   万用表打到二极管档
   红表笔接 S，黑表笔接 D
   应显示开路（OL）
   ```

2. **导通测试**
   ```
   GPIO1 输出 3.3V
   万用表打到电阻档
   测量 D-S 间电阻
   应<0.1Ω
   ```

3. **续流二极管测试**
   ```
   万用表打到二极管档
   红表笔接阴极，黑表笔接阳极
   应显示 0.6-0.7V
   反向应显示 OL
   ```

**示波器测试（可选）：**

观察 MOSFET 开关波形：
- 上升时间：<1μs
- 下降时间：<1μs
- 无振荡

### 5.6 功耗和效率

**系统功耗：**
```
待机电流：
- ESP32: 200mA (WiFi 连接)
- 传感器：20mA
- MOSFET: 0A (截止)
总计：220mA

工作电流：
- ESP32: 200mA
- 传感器：20mA
- 水泵：300mA
- MOSFET: 0.3A × 0.028Ω = 0.0025W (可忽略)
总计：520mA
```

**电池续航（单节 18650 2500mAh）：**
```
假设每天浇水 2 次，每次 30 秒：
- 工作时间：2 × 30 秒 = 1 分钟/天
- 待机电流：220mA × 23.98 小时 = 5276mAh/天
- 工作电流：520mA × 0.02 小时 = 10mAh/天

总计：5286mAh/天

续航：2500mAh / 5286mAh/天 ≈ 0.47 天

⚠️ 需要优化：
- 使用 Deep Sleep（待机电流<10μA）
- 定时唤醒浇水
- 续航可达 30 天+
```

## 8. 安全设计要点

### 8.1 温度监控

- NTC 紧贴电池表面
- 使用导热胶固定
- 远离发热元件
- 走线包地保护

### 8.2 过流保护

- 保险丝靠近电池正极
- 走线加粗（≥1.0mm）
- 便于更换（如使用插座）

### 8.3 浪涌保护

- TVS 靠近 USB 入口
- 到 GND 走线短而粗
- 5V 主电源线加粗

### 8.4 防水设计

- PCB 喷涂三防漆
- 外部接口使用防水接头
- 外壳接缝打密封胶
