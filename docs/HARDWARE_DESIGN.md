## 3. 传感器模块

### 3.1 土壤湿度传感器

**型号：** 电容式土壤湿度传感器（防水）

**参数：**
| 参数 | 值 |
|------|-----|
| 工作电压 | 3.3-5V |
| 输出信号 | 模拟 0-3.3V |
| 检测范围 | 0-100% |
| 工作电流 | <20mA |
| 耐腐蚀 | 是 |

**接线：**
```
传感器     ESP32-C3
VCC   ──── 3.3V
GND   ──── GND
AO    ──── GPIO0 (ADC)
```

**ADC 校准：**
| 状态 | ADC 值 | 湿度% |
|------|--------|-------|
| 干燥空气 | 3200 | 0% |
| 干燥土壤 | 2800 | 20% |
| 湿润土壤 | 1800 | 70% |
| 水中 | 1200 | 100% |

**计算公式：**
```cpp
moisture_percent = map(adc_value, 3200, 1200, 0, 100);
moisture_percent = constrain(moisture_percent, 0, 100);
```

---

### 3.2 水位传感器

**型号：** XKC-Y25 非接触式液位传感器

**参数：**
| 参数 | 值 |
|------|-----|
| 工作电压 | 5V |
| 输出信号 | 数字 NPN |
| 检测精度 | ±1mm |
| 响应时间 | <500ms |
| 防护等级 | IP67 |

**接线：**
```
传感器     ESP32-C3
VCC   ──── 5V
GND   ──── GND
OUT   ──── GPIO2 (INPUT_PULLUP)
                │
               10kΩ
                │
               GND
```

**逻辑：**
- 有水：OUT 高电平 (3.3V+) → GPIO2 读 HIGH
- 无水：OUT 低电平 (0V) → GPIO2 读 LOW

### 3.3 电池电压检测

**方案：** 电阻分压 + ADC

**电路：**
```
电池 (7.4-8.4V)
     │
    ┌┴┐
    │ │ R1 100kΩ
    └┬┘
     ├───┬─── GPIO3 (ADC)
     │   │
    ┌┴┐  ┌┴┐ 100nF
    │ │ R2 │ │ 滤波电容
    │ │ 200kΩ └┬┘
    └┬┘   │
     │    │
    GND  GND
```

**分压比：**
```
V_adc = V_batt × R2 / (R1 + R2)
V_adc = V_batt × 200k / 300k
V_adc = V_batt × 0.667
```

**反推电池电压：**
```cpp
float v_batt = v_adc / 0.667;
float v_batt = (adc_value / 4095.0) × 3.3 × 1.5;
```

**电压对应关系：**
| 电池电压 | ADC 值 | 电量% |
|---------|--------|-------|
| 8.4V (满) | 2800 | 100% |
| 7.4V (标称) | 2470 | 50% |
| 6.6V (低电) | 2200 | 10% |
| 6.0V (关断) | 2000 | 0% |

### 4.2 MOSFET 驱动电路

**方案：** N 沟道 MOSFET (S8050 或 AO4606)

**电路图：**
```
              +5V
               │
          ┌────┴────┐
          │  水泵   │
          │  5V     │
          └────┬────┘
               │
               │ D
               │
          ┌────┴────┐
          │  S8050  │ NPN 三极管
          │         │
GPIO1 ────┤ B     E │
    1kΩ   └────┬───┘
               │
              GND

         ┌────────────────┐
         │   1N4148       │ 续流二极管
         │  ─────────     │ (防止反电动势)
         └────────────────┘
              ││
              ││ 并联在水泵两端
              ││
```

**工作原理：**
- GPIO1 高电平 → 三极管导通 → 水泵工作
- GPIO1 低电平 → 三极管截止 → 水泵停止

**代码：**
```cpp
void pump_on() {
    digitalWrite(GPIO1, HIGH);
}

void pump_off() {
    digitalWrite(GPIO1, LOW);
}

void pump_water(int duration_ms) {
    pump_on();
    delay(duration_ms);
    pump_off();
}
```

## 7. PCB 布局建议

### 7.1 板框尺寸

**推荐尺寸：** 80×60mm (2 层板)

**布局原则：**
1. ESP32-C3 放在板子中央
2. 电源模块放在一侧（远离天线）
3. 传感器接口放在边缘
4. USB-C 放在板边

### 7.2 层叠结构

```
┌─────────────────┐  ← Top Layer (元件 + 信号)
│   元件 + 信号    │
├─────────────────┤  ← GND Plane (完整地平面)
│     GND         │
└─────────────────┘
```

### 7.3 走线规则

| 网络 | 线宽 | 说明 |
|------|------|------|
| 5V 电源 | ≥1.0mm | 电流 300mA+ |
| 3.3V 电源 | ≥0.5mm | 电流 100mA |
| GND | ≥1.0mm | 地平面 |
| 信号线 | 0.3mm | GPIO/ADC |
| 天线区域 | 禁止铺铜 | ESP32 天线下方 |

### 7.4 关键走线

**电源走线：**
- 5V 从 MP1584 到水泵：1.0mm 宽
- 3.3V 从 AMS1117 到 ESP32：0.5mm 宽
- GND 完整铺铜

**信号走线：**
- ADC 走线尽量短（土壤/电池）
- 远离电源线和 PWM 线
- 包地处理

**天线区域：**
- ESP32-C3 板载天线下方禁止铺铜
- 保持 5mm 净空区
- 不要走电源线
