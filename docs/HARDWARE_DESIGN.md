## 3. 传感器模块

### 3.1 电池温度传感器 ⭐ P0 安全关键

**型号：** NTC 热敏电阻 10kΩ (B3950)

**电路原理图：**
```
         3.3V
          │
         ┌┴┐
         │ │ R1 10kΩ 上拉
         └┬┘
          │
          ├───┬─── GPIO3 (NTC_Temp)
          │   │
         ┌┴┐ ┌┴┐ C1 100nF
         │ │ │ │ 滤波电容
         └┬┘ └┬┘
          │   │
          │  GND
          │
         ┌┴┐
         │ │ TH1 NTC 10kΩ B3950
         └┬┘
          │
         GND
```

**参数：**
| 参数 | 值 |
|------|-----|
| 标称阻值 | 10kΩ @ 25°C |
| B 值 | 3950K |
| 精度 | ±1% |
| 工作温度 | -40~125°C |

**安装位置：**
- 紧贴电池组中间（两节电池之间）
- 使用导热胶固定
- 远离发热元件（MOSFET、稳压器）

**温度计算：**
```cpp
float read_battery_temp() {
    int adc = analogRead(GPIO3);
    float resistance = 10000.0 * (4095.0 / adc - 1);
    float steinhart = 1 / (0.003354016 + 0.0003354016 * log(resistance));
    return steinhart - 273.15;  // 转换为°C
}
```

**保护阈值：**
| 温度 | 动作 |
|------|------|
| ≥60°C | 立即切断充电和负载，蜂鸣器报警 |
| ≥50°C | 降低充电电流，发送警告通知 |
| ≤0°C | 禁止充电（低温充电会损坏电池） |
| ≤-10°C | 切断负载（低温放电容量极低） |

**⚠️ 安全关键：** 此传感器为 P0 级安全配置，不可省略！

## 5. 执行器模块

### 5.1 水泵驱动（MOSFET）

**电路：**
```
              5V
               │
          ┌────┴────┐
          │  水泵   │
          │  5V     │
          └────┬────┘
               │
               │ D
               │
          ┌────┴────┐
          │  AO4606 │ N-MOSFET ⭐
          │         │
GPIO1 ────┤ G     S │
    100Ω  └────┬───┘
               │
              GND
         
         ┌────────────────┐
         │   1N4148       │ 续流二极管
         │  ─────────     │ (防止反电动势)
         └────────────────┘
              ││
              ││ 并联在水泵两端
              ││
```

**元件选择：**
| 元件 | 型号 | 说明 |
|------|------|------|
| MOSFET | AO4606 | N 沟道，Vds=30V，Id=6.9A |
| 续流二极管 | 1N4148 | 防止反电动势 |
| 栅极电阻 | 100Ω | 限制栅极电流 |

**工作原理：**
- GPIO1 高电平 → MOSFET 导通 → 水泵工作
- GPIO1 低电平 → MOSFET 截止 → 水泵停止

## 8. 安全设计要点

### 8.1 温度监控

- NTC 紧贴电池表面
- 使用导热胶固定
- 远离发热元件
- 走线包地保护

### 8.2 过流保护

- 保险丝靠近电池正极
- 走线加粗（≥1.0mm）
- 便于更换（如使用插座）

### 8.3 浪涌保护

- TVS 靠近 USB 入口
- 到 GND 走线短而粗
- 5V 主电源线加粗
