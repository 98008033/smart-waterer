---

### Day 9-10：焊接装配

**焊接顺序：**
1. [ ] 矮元件（电阻、电容）
2. [ ] 高元件（端子、连接器）
3. [ ] 保险丝、TVS
4. [ ] NTC（带引线）
5. [ ] 保护板
6. [ ] 电池座

**注意事项：**
- ⚠️ 二极管极性不要焊反
- ⚠️ NTC 引线留长（贴电池用）
- ⚠️ 保险丝方向正确

---

### Day 11：喷涂三防漆

**操作步骤：**
1. [ ] 清洁 PCB（无水酒精）
2. [ ] 遮蔽连接器（美纹纸）
3. [ ] 均匀喷涂三防漆
4. [ ] 固化 24 小时（或 UV 灯）
5. [ ] 去除遮蔽纸

**注意：**
- 三防漆不要喷到连接器
- 通风环境操作
- 戴手套和口罩

---

### Day 12-13：功能测试

#### 电气测试
- [ ] 万用表测短路（电池输入）
- [ ] 上电测电压（5V/3.3V）
- [ ] 静态电流（应<50mA）
- [ ] 工作电流（水泵工作时）

#### 温度保护测试
- [ ] 室温 NTC 读数（20-30°C）
- [ ] 手捏 NTC 温度上升
- [ ] 吹风机加热到 50°C（应限制充电）
- [ ] 加热到 60°C（应切断充放电）
- [ ] 冷却后自动恢复

#### 过流保护测试
- [ ] 正常负载电流
- [ ] 短路输出（保险丝应跳闸）
- [ ] 冷却后自动恢复

#### 防水测试
- [ ] 外壳组装
- [ ] 接缝打密封胶
- [ ] 淋雨测试 10 分钟
- [ ] 打开检查无进水

---

### Day 14：环境测试

#### 高温测试
```
条件：45°C 环境（高温箱或夏天阳台）
时间：连续 24 小时
监测：
- 电池温度（应<50°C）
- 充电电流（高温时应限制）
- 设备功能（应正常）
```

#### 低温测试（可选）
```
条件：0°C 环境（冰箱冷藏室）
时间：4 小时
监测：
- 电池温度（应>0°C）
- 充电禁止（低温保护）
```

#### 湿度测试
```
条件：90% RH（密封盒 + 湿毛巾）
时间：48 小时
监测：
- PCB 无凝露
- 功能正常
```
