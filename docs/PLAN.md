### 4.4 收货检查

**保护板测试：**
```
万用表测量：
1. 电池接上后，P+ P- 应有电压
2. 短路 P+ P-，保护板应切断（电压为 0）
3. 加热 NTC（打火机烤），60°C 时应切断
4. 冷却后应自动恢复
```

**注意事项：**
- ⚠️ 保护板必须带温度保护！
- ⚠️ NTC 要买 B3950 值
- ⚠️ 三防漆买含氟的
- ⚠️ 密封胶买 704 型号
