## 第 1 步：注册账号（10 分钟）

### 1.1 腾讯云账号

```
1. 访问：https://cloud.tencent.com
2. 点击"免费注册"
3. 选择"个人"账号
4. 微信扫码登录
5. 完成实名认证（需要身份证）
```

**⏰ 耗时：** 5 分钟

### 1.2 立创 EDA 账号

```
1. 访问：https://lceda.cn
2. 点击"注册"
3. 选择 QQ/微信登录
4. 完成实名认证
```

**⏰ 耗时：** 3 分钟

## 第 2 步：克隆开源项目（10 分钟）

### 2.1 搜索 ESP32-C3 项目

```
1. 访问：https://lceda.cn/project
2. 搜索框输入："ESP32-C3"
3. 按"最多克隆"排序
4. 选择评分高的项目
```

**推荐项目：**
- ESP32-C3 核心板
- ESP32-C3 最小系统
- ESP32 开发板

### 2.2 克隆项目

```
1. 打开找到的项目
2. 点击右上角"克隆到编辑器"
3. 选择"原理图"和"PCB"
4. 等待克隆完成
5. 自动打开编辑器
```

**⏰ 耗时：** 5 分钟

### 2.3 熟悉界面

```

│ 菜单栏                          │
├─────────┬───────────┬───────────┤
│ 元件库  │  原理图   │  属性面板 │
│         │  编辑器   │           │
├─────────┴───────────┴───────────┤
│  状态栏 / 图层切换                 │
└─────────────────────────────────┘
```

**⏰ 耗时：** 5 分钟

## 第 3 步：修改设计（30 分钟）

### 3.1 添加传感器接口（10 分钟）

**土壤湿度接口：**
```
1. 在元件库搜索"3Pin 排针"
2. 拖放到原理图
3. 标注："土壤湿度"
4. 连接：
   - Pin1 → 3.3V
   - Pin2 → GND
   - Pin3 → GPIO0
```

**水位检测接口：**
```
1. 再放置一个 3Pin 排针
2. 标注："水位检测"
3. 连接：
   - Pin1 → 3.3V
   - Pin2 → GND
   - Pin3 → GPIO2
```

### 3.2 添加水泵驱动（10 分钟）

```
1. 搜索"S8050"三极管
2. 放置到原理图
3. 基极 → GPIO1 (串联 1kΩ电阻)
4. 发射极 → GND
5. 集电极 → 水泵负极
6. 水泵正极 → 5V
7. 并联 1N4148 二极管（反向）
```

### 3.3 添加电源模块（10 分钟）

**如果开源项目已有电源：**
- 检查是否有 5V 和 3.3V 输出
- 检查是否有电池接口
- 如需要，添加 TP5100 充放电模块

**如果需要添加：**
```
1. 搜索"TP5100"
2. 放置到原理图
3. 连接：
   - IN+ → USB-C 5V
   - OUT+ → 电池正极
   - 电池负极 → GND
```

## 第 4 步：下单打样（10 分钟）

### 4.1 设计规则检查

```
1. 点击"工具" → "设计规则检查"
2. 等待检查完成
3. 修复所有错误和警告
4. 确认无错误
```

**⏰ 耗时：** 3 分钟

### 4.2 生成 Gerber 文件

```
1. 点击"制造" → "PCB 打样"
2. 自动打开嘉立创页面
3. 检查参数：
   - 尺寸：80×60mm
   - 层数：2 层
   - 板厚：1.6mm
   - 铜厚：1oz
   - 阻焊：绿色
   - 数量：5 片
```

**⏰ 耗时：** 3 分钟

### 4.3 下单付款

```
1. 点击"添加到购物车"
2. 登录嘉立创账号（用立创 EDA 账号）
3. 使用新人券（首单免费）
4. 提交订单
5. 等待收货（3-5 天）
```

**⏰ 耗时：** 4 分钟

## ✅ 完成！下一步做什么？

### 等待 PCB 期间（3-5 天）

**可以做的事：**

1. **采购元件**
   - 访问立创商城：https://www.szlcsc.com
   - 上传 BOM 文件
   - 下单付款
   - 参考：[BOM 清单](BOM.md)

2. **学习固件开发**
   - 安装 Arduino IDE
   - 学习 ESP32 编程
   - 参考：[固件开发指南](FIRMWARE_DEVELOPMENT.md)

3. **配置云平台**
   - 注册腾讯云 IoT
   - 创建产品
   - 创建设备
   - 参考：[腾讯云接入指南](TENCENT_IOT_SETUP.md)

### PCB 到货后

1. **焊接组装**（2-3 小时）
   - 焊接元件
   - 连接传感器
   - 装入防水盒

2. **烧录固件**（30 分钟）
   - 连接 USB
   - 烧录代码
   - 配置 WiFi

3. **测试验证**（1 小时）
   - 上电测试
   - 连接小程序
   - 测试浇水功能

## 🆘 遇到问题？

### 常见问题速查

| 问题 | 解决方案 |
|------|---------|
| 立创 EDA 打不开 | 用 Chrome 浏览器，清除缓存 |
| 元件库找不到 | 使用"万能库"搜索 |
| DRC 检查报错 | 查看错误位置，修复后重试 |
| PCB 打样失败 | 检查板框是否封闭 |
| 元件买不到 | 找替代品或淘宝采购 |
