## 1. 产品概述

### 1.1 产品定位
基于 ESP32 的智能家居浇花系统，支持手机 App/ThingsCloud Dashboard 远程监控和自动控制。

### 1.2 目标用户
- 经常出差/旅行的植物爱好者
- 办公室绿植养护人员
- 阳台/庭院种植用户

### 1.3 核心价值
- ✅ 自动浇水，不用担心植物干死
- ✅ 远程监控，随时查看植物状态
- ✅ 智能报警，缺水/低电/高温及时通知
- ✅ **安全保障，温度监控 + 多重保护** ⭐

### 1.4 技术选型
- **IoT 平台：** ThingsCloud（开源，免费）⭐
- **主控：** ESP32-C3（WiFi+BLE）
- **安全：** NTC 温度监控 + 保护板 + 保险丝

## 4. 用户体验

### 4.1 设备初始化流程

**首次配网：**
```
1. 设备通电 → LED 快闪 (配网模式)
2. 手机打开 Dashboard/小程序 → 搜索设备
3. 选择 WiFi → 输入密码
4. Dashboard 发送 WiFi 配置
5. 设备连接 WiFi → LED 慢闪 → 常亮 (成功)
6. 配网完成
```

**重置配网：**
```
长按重置键 5 秒 → 恢复出厂设置 → LED 快闪 → 重新配网
```

### 4.2 Dashboard/App 界面

```
┌─────────────────────────────────┐
│     🌱 智能浇花机               │
├─────────────────────────────────┤
│  [💧 手动浇水]                  │
│  [自动/手动]                    │
│                                 │
│  浇水频率：12 小时              │
│  浇水时长：2000ms               │
│                                 │
│  土壤湿度：75%                  │
│  水箱水位：正常                 │
│  电池电压：8.2V                 │
│  **电池温度：35°C** ✅          │
│                                 │
│  浇水历史：累计 15 次            │
└─────────────────────────────────┘
```

### 4.3 状态指示

| LED 状态 | 含义 |
|---------|------|
| 慢闪 (1s) | 正常工作 |
| 快闪 (100ms) | WiFi 连接中 |
| 常亮 | 正在浇水 |
| **双闪** | **报警状态（高温/过流）** ⭐ |
| 熄灭 | 休眠/关机 |

### 4.4 安全保护机制

**温度保护：**
```
正常 (0-45°C)   → 正常工作
偏高 (45-50°C)  → 发送警告通知
高温 (50-60°C)  → 限制充电，降流
紧急 (≥60°C)    → 切断充放电，蜂鸣器报警
极低温 (≤-10°C) → 切断负载
```

**过流保护：**
```
正常 (<1A)      → 正常工作
过载 (1-2A)     → 保险丝限流
短路 (≥2A)      → 保险丝跳闸切断
```

**电压保护：**
```
满电 (8.4V)     → 停止充电
正常 (7.4V)     → 正常工作
低压 (6.6V)     → 发送低电警告
空电 (6.0V)     → 切断负载
```

### 6.1 技术风险

## 8. 安全规范 ⭐

### 8.1 电池安全

**必须满足：**
1. NTC 温度传感器实时监控
2. 2 串锂电保护板（带温度保护）
3. 1A 自恢复保险丝
4. TVS 浪涌保护
5. 60°C 切断充放电
6. 0°C 禁止充电
7. -10°C 切断负载

### 8.2 防水安全

**必须满足：**
1. PCB 喷涂三防漆
2. 外壳接缝打密封胶
3. 外部接口使用防水接头
4. IP65 淋雨测试通过

### 8.3 使用安全

**用户须知：**
1. ⚠️ 禁止阳光直射
2. ⚠️ 禁止密闭空间
3. ✅ 阴凉通风处使用
4. ✅ 定期检查电池状态
