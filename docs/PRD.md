## 1. 产品概述

### 1.1 产品定位
基于 ESP32 的智能家居浇花系统，支持手机 App/ThingsCloud Dashboard 远程监控和自动控制。

### 1.2 目标用户
- 经常出差/旅行的植物爱好者
- 办公室绿植养护人员
- 阳台/庭院种植用户

### 1.3 核心价值
- ✅ 自动浇水，不用担心植物干死
- ✅ 远程监控，随时查看植物状态
- ✅ 智能报警，缺水/低电/高温及时通知
- ✅ **安全保障，温度监控 + 多重保护** ⭐
- ✅ **成本优化，简化非核心功能** ⭐

### 1.4 技术选型
- **IoT 平台：** ThingsCloud（开源，免费）⭐
- **主控：** ESP32-C3（WiFi+BLE）
- **安全：** NTC 温度监控 + 保护板 + 保险丝
- **供电：** TP4056 + 单节 18650 ⭐

### 4.4 安全保护机制

**温度保护：**
```
正常 (0-45°C)   → 正常工作
偏高 (45-50°C)  → 发送警告通知
高温 (50-60°C)  → 限制充电，降流
紧急 (≥60°C)    → 切断充放电
极低温 (≤-10°C) → 切断负载
```

**过流保护：**
```
正常 (<1A)      → 正常工作
过载 (1-2A)     → 保险丝限流
短路 (≥2A)      → 保险丝跳闸切断
```

**电压保护：**
```
满电 (4.2V)     → 停止充电
正常 (3.7V)     → 正常工作
低压 (3.5V)     → 发送低电警告
空电 (3.0V)     → 切断负载
```

## 8. 安全规范 ⭐

### 8.1 电池安全

**必须满足：**
1. NTC 温度传感器实时监控
2. 单节锂电保护板（带温度保护）
3. 1A 自恢复保险丝
4. TVS 浪涌保护
5. 60°C 切断充放电
6. 0°C 禁止充电
7. -10°C 切断负载

### 8.2 防水安全

**必须满足：**
1. PCB 喷涂三防漆
2. 外壳接缝打密封胶
3. 外部接口使用防水接头
4. IP65 淋雨测试通过

### 8.3 使用安全

**用户须知：**
1. ⚠️ 禁止阳光直射
2. ⚠️ 禁止密闭空间
3. ✅ 阴凉通风处使用
4. ✅ 定期检查电池状态

### 8.4 采购规范

**关键元件采购要求：**

| 元件 | 规格要求 | 验证方法 |
|------|---------|---------|
| NTC 热敏电阻 | B 值 3950，25°C 时 10kΩ | 万用表测阻值 |
| 保护板 | 带温度保护，60°C 切断 | 打火机加热测试 |
| 保险丝 | 1A 自恢复，1812 封装 | 短路测试 |
| 三防漆 | 含氟，防潮防水 | 喷涂后滴水测试 |
| 密封胶 | 704 硅橡胶，耐高温 | 固化后拉伸测试 |

**禁止购买：**
- ❌ 不带温度保护的保护板
- ❌ NTC B 值不是 3950
- ❌ 一次性保险丝（应自恢复）
- ❌ 普通胶水代替密封胶

### 9.3 常见问题

**Q: 保护板不工作？**  
**A:** 检查电池接线是否正确，用万用表测量输出电压。

**Q: NTC 读数不准？**  
**A:** 确认 B 值是 3950，检查上拉电阻是否为 10kΩ。

**Q: 保险丝频繁跳闸？**  
**A:** 检查负载电流是否超过 1A，或更换 1.5A 规格。

**Q: 防水失效？**  
**A:** 检查密封胶是否固化完全，接头是否拧紧。
