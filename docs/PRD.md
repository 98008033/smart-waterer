# 智能浇花机 - 产品需求文档 (PRD)

**版本：** v0.1
**日期：** 2026-03-12
**状态：** 待评审

---

## 1. 产品概述

### 1.1 产品定位
基于 ESP32 的智能家居浇花系统，支持手机 App 远程监控和自动控制。

### 1.2 目标用户
- 经常出差/旅行的植物爱好者
- 办公室绿植养护人员
- 阳台/庭院种植用户

### 1.3 核心价值
- ✅ 自动浇水，不用担心植物干死
- ✅ 远程监控，随时查看植物状态
- ✅ 智能报警，缺水/低电及时通知

### 2.3 P2 可选功能 (可以有)

| ID | 功能 | 描述 | 优先级 |
|----|------|------|--------|

## 3. 技术指标

### 3.1 性能指标

| 指标 | 目标值 | 说明 |
|------|--------|------|
| 续航时间 | ≥30 天 | 电池供电 |
| 浇水精度 | ±5ml | 水泵控制 |
| 响应延迟 | <3 秒 | App 控制 |
| 土壤精度 | ±5% | 湿度检测 |
| 防水等级 | IP65 | 户外使用 |

### 3.2 硬件规格

| 项目 | 规格 |
|------|------|
| 主控 | ESP32-C3 SuperMini |
| 电源 | 18650×2 (7.4V 5000mAh) |
| 水泵 | DC5V 微型水泵 |
| 土壤传感器 | 电容式 (防水) |
| 水位传感器 | 非接触式 |
| 外壳 | IP65 防水盒 |

### 3.3 软件规格

| 项目 | 规格 |
|------|------|
| 固件 | Arduino + ESP-IDF |
| App | Blynk (初期) / 微信小程序 (后期) |
| 通信 | WiFi 2.4GHz + BLE 5.0 |
| OTA | 支持固件升级 |

## 4. 用户体验

### 4.0 设备初始化流程

**首次配网：**
```
1. 设备通电 → LED 快闪 (配网模式)
2. 手机打开 App → 搜索设备
3. 选择 WiFi → 输入密码
4. App 通过蓝牙发送 WiFi 配置
5. 设备连接 WiFi → LED 慢闪 → 常亮 (成功)
6. 配网完成
```

**重置配网：**
```
长按重置键 5 秒 → 恢复出厂设置 → LED 快闪 → 重新配网
```

**配网方式：**
- 蓝牙配网 (BLE 5.0) - 推荐
- SmartConfig (WiFi 配网)
- AP 模式 (设备创建热点)
