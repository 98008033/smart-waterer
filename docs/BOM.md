## 📋 收货检查

### 立创商城
- [ ] 元件数量正确
- [ ] 型号与订单一致
- [ ] 引脚无氧化
- [ ] 包装完好

### 淘宝（保护板）
- [ ] 保护板功能测试（万用表）
- [ ] NTC 加热 60°C 应切断
- [ ] 短路输出应保护
- [ ] 冷却后自动恢复

### 保护板测试方法
```
万用表测量：
1. 电池接上后，P+ P- 应有电压（7.4-8.4V）
2. 短路 P+ P-，保护板应切断（电压为 0）
3. 用打火机烤 NTC，60°C 时应切断
4. 冷却后应自动恢复
```

### 淘宝（防水配件）
- [ ] 防水接头尺寸正确（PG7）
- [ ] 密封胶未过期
- [ ] 防水盒无裂纹
- [ ] 密封圈完好

---

## ⚠️ 注意事项

1. **保护板必须带温度保护！** 购买时确认商品描述
2. **NTC 要买 B3950 值**，其他 B 值测温不准
3. **三防漆买含氟的**，防潮效果更好
4. **密封胶买 704 型号**，耐高温绝缘
5. **防水盒确认尺寸**，要能装下 PCB 和电池
