### 淘宝
- [ ] 保护板功能正常（万用表测试）

### 保护板测试
```
用万用表测量：
1. 电池接上后，输出应有电压
2. 短路输出，保护板应切断（电压为 0）
3. 加热 NTC（打火机烤），60°C 时应切断
```
