你提出的问题非常关键！经过全面审查，发现原设计存在以下严重安全隐患：

### ⚠️ 核心问题：电池无温度监控

**风险场景：**
```
夏天正午 (环境 45°C) + 阳光直射 (外壳 55°C) + 充电发热 (10°C)
    ↓
电池温度 65°C+ → 隔膜融化 → 内部短路 → 热失控 🔥
```

**18650 锂电池温度特性：**
| 温度 | 影响 |
|------|------|
| 0-45°C | ✅ 正常工作 |
| 45-60°C | ⚠️ 容量衰减加速 |
| 60-70°C | 🔴 SEI 膜分解，危险 |
| 70-90°C | 🔴 隔膜收缩，极危险 |
| 90°C+ | 💥 隔膜融化，热失控 |

## 🔧 硬件改动

### 1. 温度监控电路

```
电池组 ──→ NTC 10kΩ (B3950) ──→ GPIO3 (ADC)
              │
             10kΩ 上拉
              │
             GND
```

### 2. 保护板电路

```
18650×2 ──→ 2 串保护板 (带温度) ──→ 保险丝 ──→ TP5100
                                      │
                                    1A 自恢复
```

### 3. 浪涌保护

```
USB-C 5V ──→ SMAJ5.0A TVS ──→ TP5100
                  │
                 GND
```

## 💻 固件改动

### 新增模块：`battery_safety.c`

**功能：**
- NTC 温度读取（Steinhart-Hart 方程）
- 温度保护逻辑（60°C 切断，50°C 警告）
- 电压保护逻辑（8.4V 过充，6.0V 过放）
- 状态回调通知

**保护阈值：**
```c
#define TEMP_CRITICAL_HIGH  60.0f   // 紧急切断
#define TEMP_WARNING_HIGH   50.0f   // 警告
#define TEMP_CRITICAL_LOW   0.0f    // 禁止充电
#define TEMP_EXTREME_LOW    -10.0f  // 切断负载

#define VOLTAGE_FULL        8.40f   // 满电
#define VOLTAGE_EMPTY       6.00f   // 空电
```
