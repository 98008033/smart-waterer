## 1. 🔴 P0 严重风险（必须整改）

### 1.1 电池无温度监控 ⚠️⚠️⚠️

**风险描述：**
- 18650 锂电池在充放电过程中会产生热量
- 夏季阳台/室外环境温度可达 40-50°C
- 电池表面温度可能达到 60°C+
- 无温度监控可能导致热失控、起火、爆炸

**风险场景：**
```
夏天正午 (环境温度 45°C)
    ↓
设备在阳光直射下 (外壳温度 55°C)
    ↓
电池充电中 (发热 +10°C)
    ↓
电池温度 65°C+ → 隔膜融化 → 内部短路 → 热失控 🔥
```

**18650 温度特性：**
| 温度范围 | 影响 | 安全等级 |
|---------|------|---------|
| 0-45°C | 正常工作 | ✅ 安全 |
| 45-60°C | 容量衰减加速 | ⚠️ 注意 |
| 60-70°C | SEI 膜分解 | 🔴 危险 |
| 70-90°C | 隔膜收缩 | 🔴 极危险 |
| 90°C+ | 隔膜融化、内部短路 | 💥 热失控 |

**整改方案：**

#### 方案 A：NTC 温度传感器（推荐）
```
电池组 ──┬── NTC 热敏电阻 ──→ GPIO (ADC)
          │
          └── 10kΩ上拉 ──→ 3.3V
```

**元件：**
- NTC 热敏电阻 10kΩ (B3950) - ¥0.5
- 贴片在电池组中间

**代码逻辑：**
```cpp
#define NTC_PIN GPIO3
#define TEMP_CRITICAL 60  // 60°C 切断充电
#define TEMP_WARNING 50   // 50°C 降流充电

float read_battery_temp() {
    int adc = analogRead(NTC_PIN);
    float resistance = 10000.0 * (4095.0 / adc - 1);
    // Steinhart-Hart 方程
    float temp = 1 / (0.003354016 + 0.0003354016 * log(resistance));
    return temp - 273.15;  // 转换为°C
}

void battery_safety_check() {
    float temp = read_battery_temp();
    
    if (temp >= TEMP_CRITICAL) {
        // 紧急切断充电和放电
        disable_charging();
        disable_load();
        beep_alert();
        send_alert("电池过热！");
    }
    else if (temp >= TEMP_WARNING) {
        // 降低充电电流
        reduce_charge_current();
        send_warning("电池温度偏高");
    }
}
```

**成本：** ¥1-2  
**实施难度：** 低  
**优先级：** P0

---

#### 方案 B：DS18B20 数字温度传感器
```
电池组 ──→ DS18B20 ──→ GPIO (单总线)
```

**优点：** 精度高 (±0.5°C)  
**缺点：** 成本较高 (¥3-5)，需要额外 GPIO

#### 方案 C：带温度保护的电池保护板
```
18650×2 + 保护板 (带温度保护)
```

**保护板功能：**
- 过充保护 (4.25V)
- 过放保护 (2.75V)
- 过流保护 (3A)
- **温度保护 (60°C 切断)**

**成本：** ¥15-20 (含温度保护)  
**优先级：** P0

### 1.2 无防水设计 ⚠️⚠️⚠️

**风险描述：**
- 浇花机工作环境潮湿
- 水可能进入电路导致短路
- 短路可能引发火灾或设备损坏

**风险场景：**
```
浇水时水溅入外壳
    ↓
PCB 短路
    ↓
电池短路 → 发热/起火 🔥
或 ESP32 损坏
```

**整改方案：**

#### 必须措施：
1. **IP65 防水盒** - 已有规划 ✅
2. **防水接头** - 所有外部接口使用防水接头
3. **三防漆** - PCB 喷涂三防漆 (防潮、防盐雾、防霉)
4. **密封胶** - 外壳接缝处打密封胶

#### 接线防水方案：
```
方案 1: PG7 防水葛兰头
  外壳 ──→ PG7 接头 ──→ 传感器线
  
方案 2: 防水航空插头
  外壳 ──→ 4Pin 航空插头 ──→ 传感器
  
方案 3: 灌封胶
  接线端子处灌封硅胶
```

**成本：** ¥10-20  
**优先级：** P0

---

### 1.3 无过流/短路保护 ⚠️⚠️⚠️

**风险描述：**
- 水泵短路可能引起电池过流
- 电线短路可能导致火灾
- 当前设计无保险丝

**整改方案：**

#### 添加保险丝：
```
电池+ ──→ 1A 自恢复保险丝 ──→ 电源输入
```

**元件选择：**
| 类型 | 型号 | 保持电流 | 跳闸电流 | 成本 |
|------|------|---------|---------|------|
| 自恢复 | 1812L110 | 1.1A | 2.2A | ¥0.5 |
| 自恢复 | 1812L150 | 1.5A | 3A | ¥0.5 |
| 一次性 | 3.6×10mm 1A | 1A | - | ¥0.2 |

**推荐：** 自恢复保险丝 (故障排除后自动恢复)

**成本：** ¥0.5-1  
**优先级：** P0

## 2. 🟠 P1 中等风险（建议整改）

### 2.1 电池过放保护仅靠固件 ⚠️⚠️

**风险描述：**
- 当前过放保护依赖 ESP32 固件
- 如果 ESP32 死机/故障，电池可能过放
- 锂电池过放会导致永久损坏

**整改方案：**

#### 添加硬件保护板（强烈推荐）：
```
18650×2 ──→ 2 串锂电池保护板 ──→ 输出
```

**保护板参数：**
- 过充保护：4.25V ±0.05V
- 过放保护：2.75V ±0.1V
- 过流保护：3A
- 短路保护：有

**成本：** ¥8-15  
**优先级：** P1

---

### 2.2 无防雷击/浪涌保护 ⚠️⚠️

**风险描述：**
- USB-C 充电时可能遭遇雷击浪涌
- 电网波动可能损坏充电电路

**整改方案：**

#### 添加 TVS 二极管：
```
USB-C 5V ──→ SMAJ5.0A TVS ──→ TP5100
                    │
                   GND
```

**元件：** SMAJ5.0A (单向 TVS)  
**成本：** ¥0.5-1  
**优先级：** P1

---

## 3. 🟡 P2 低风险（可选优化）

### 3.1 电池反接保护

**风险：** 电池接反可能损坏电路  
**方案：** 串联肖特基二极管 (1N5819)  
**成本：** ¥0.2

---

### 3.2 静电防护 (ESD)

**风险：** 人体静电可能损坏 ESP32  
**方案：** 传感器接口添加 TVS 阵列 (SM712)  
**成本：** ¥1-2

---

### 3.3 外壳接地

**风险：** 漏电可能电击用户  
**方案：** 金属外壳接地螺丝  
**成本：** ¥1

| 地区 | 夏季最高温 | 设备表面温度 | 电池温度 (无散热) |
|------|-----------|-------------|------------------|
| 广州/深圳 | 35-38°C | 50-55°C | 60-70°C 🔴 |

| 室内空调房 | 25-28°C | 30-35°C | 35-45°C ✅ |

### 4.2 安全建议

#### 安装位置要求：
```
✅ 推荐：
- 室内阳台 (避免阳光直射)
- 阴凉通风处
- 有遮挡的室外

❌ 禁止：
- 阳光直射
- 密闭空间
- 靠近热源
```

#### 夏季使用限制：
```cpp
// 夏季模式 (环境温度>35°C)
if (ambient_temp > 35) {
    // 禁止充电
    disable_charging();
    
    // 限制工作时间
    max_pump_duration = 30 秒;
    
    // 增加散热间隔
    cooldown_interval = 10 分钟;
}
```
