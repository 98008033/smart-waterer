### 3.3 电池电量检测

**方案：分压电阻 + ADC**

```
电池 (3.7V) ──┬── R1 100k ──┬── GPIO3 (ADC)
              │             │
              └── R2 200k ──┴── GND

分压比 = R2 / (R1+R2) = 200/300 = 0.667
ADC 电压 = 电池电压 × 0.667

电池 4.2V → ADC 2.8V (3300)
电池 3.7V → ADC 2.47V (2900)
电池 3.3V → ADC 2.2V (2600) ← 低电报警
```

**成本：** 电阻 ¥0.5

**续航计算：**
```
待机电流：50μA (Deep Sleep)
WiFi 连接：80mA (10 秒/次)
水泵工作：300mA (5 秒/次)
每天 2 次浇水

       = 1.2mAh + 0.44mAh + 0.83mAh

续航 = 5000mAh / 2.5mAh/天 = 2000 天 (理论)
实际续航 ≈ 30-45 天 (考虑自放电 + 温度)
```

### 8.2 低功耗策略

```cpp
// 工作周期
void loop() {
    wake_up();           // 唤醒
    connect_wifi();      // 连接 WiFi (5 秒)
    read_sensors();      // 读取传感器 (1 秒)
    send_data();         // 发送数据 (2 秒)
    check_schedule();    // 检查浇水计划
    if (need_water) {
        water_plant();   // 浇水 (5 秒)
    }
    deep_sleep(30min);   // 深睡 30 分钟
}

// 功耗分析

水泵工作      300mA    5s×2       0.008mAh
─────────────────────────────────────────
总计                                   ~1.7mAh/天
```

### 8.3 浇水逻辑

```cpp
// 自动浇水条件
if (soil_moisture < threshold_dry) {     // 土壤干燥
    if (water_level == HIGH) {           // 水箱有水
        if (battery > 20%) {             // 电量充足
            water_plant();               // 执行浇水
        } else {
            send_alert("电池电量低");      // 低电报警
        }
    } else {
        send_alert("水箱缺水");           // 缺水报警
    }
}

// 浇水函数
void water_plant() {
    pump_on();
    delay(water_duration);  // 根据水量设定
    pump_off();
    log_water_event();      // 记录浇水历史
    send_notification();    // 推送通知
}
```
