### 3.3 引脚连接表

**ESP32-C3 引脚分配：**

| GPIO | 功能 | 连接 | 说明 |
|------|------|------|------|
| GPIO0 | 土壤湿度 | 3Pin 排针 AO | ADC 输入 |
| GPIO1 | 水泵控制 | S8050 基极 | PWM 输出 |
| GPIO2 | 水位检测 | 3Pin 排针 OUT | 数字输入 |
| GPIO3 | 电池电压 | 分压电阻 | ADC 输入 |
| GPIO4 | LED 指示 | LED+ 电阻 | 数字输出 |
| GPIO5 | 蜂鸣器 | 三极管基极 | 数字输出 |
| 3.3V | 传感器供电 | 3Pin 排针 VCC | 电源输出 |

| GND | 公共地 | 所有 GND | 公共地 |

### 3.4 原理图绘制步骤

**第 1 步：放置 ESP32-C3**
```
1. 在元件库搜索"ESP32-C3 SuperMini"
2. 拖放到原理图中央
3. 旋转 90 度（引脚朝右）
```

**第 2 步：添加传感器接口**
```
1. 放置 3Pin 排针×2
2. 标注：土壤湿度、水位检测
3. 连接 VCC → 3.3V
4. 连接 GND → GND
5. 连接 AO → GPIO0
6. 连接 OUT → GPIO2
```

**第 3 步：添加水泵驱动**
```
1. 放置 S8050 三极管
2. 基极 → GPIO1 (串联 1kΩ电阻)
3. 发射极 → GND
4. 集电极 → 水泵负极
5. 水泵正极 → 5V
6. 并联 1N4148 二极管（反向）
```

**第 4 步：添加电池检测**
```
1. 放置 100kΩ和 200kΩ电阻串联
2. 上端 → 电池正极
3. 下端 → GND
4. 中间抽头 → GPIO3
5. 并联 100nF 电容滤波
```

**第 5 步：添加 LED 和蜂鸣器**
```
LED:
1. LED 正极 → 3.3V (串联 220Ω电阻)
2. LED 负极 → GPIO4

蜂鸣器:
1. 蜂鸣器正极 → 3.3V
2. 蜂鸣器负极 → 三极管集电极
3. 三极管基极 → GPIO5 (串联 1kΩ电阻)
4. 三极管发射极 → GND
```

**第 6 步：添加电源模块**
```
TP5100:
1. IN+ → USB-C 5V
2. OUT+ → 电池正极
3. 电池负极 → GND

MP1584:
1. VIN → 电池正极
2. VOUT → 5V
3. GND → GND

AMS1117-3.3:
1. VIN → 5V
2. VOUT → 3.3V
3. GND → GND
4. 输入输出各加滤波电容
```

## 4. PCB 布局指南

### 4.1 板框尺寸

**推荐尺寸：** 80×60mm

**设置方法：**
1. 点击"设计" → "板框"
2. 选择"自定义板框"
3. 绘制 80×60mm 矩形
4. 确认

### 4.2 布局原则

**重要规则：**
1. ESP32-C3 放在板子中央
2. 电源模块放在一侧（远离天线）
3. 传感器接口放在边缘（方便接线）
4. USB-C 放在板边（方便充电）
5. 天线区域保持净空（不要铺铜）

**布局顺序：**
```
1. 先放置 ESP32-C3（中央）
2. 放置电源模块（左下角）
3. 放置传感器接口（上边缘）
4. 放置水泵接口（右边缘）
5. 放置 LED 和蜂鸣器（角落）
```

### 4.3 推荐布局

```
┌────────────────────────────────────────┐
│  ○ 安装孔                        USB-C │
│                                        │
│   ┌──────────────┐                     │
│   │   传感器接口  │   ESP32-C3        │
│   │ (土壤/水位)  │   SuperMini       │
│   └──────────────┘                     │
│                                        │
│   ┌──────┐                    ┌──────┐│
│   │ LED  │                    │水泵  ││
│   └──────┘                    │接口  ││
│                                └──────┘│
│   ┌──────┐                             │
│   │蜂鸣器│    电源模块                 │
│   └──────┘   TP5100+MP1584            │
│                                电池接口 │
│  ○ 安装孔                              │
└────────────────────────────────────────┘
```

### 4.4 布线规则

**电源走线：**
| 网络 | 线宽 | 说明 |
|------|------|------|
| 5V | ≥1.0mm | 电流 300mA+ |
| 3.3V | ≥0.5mm | 电流 100mA |
| GND | ≥1.0mm | 地平面 |
| 电池 | ≥1.0mm | 电流 2A |

**信号走线：**
| 网络 | 线宽 | 说明 |
|------|------|------|
| ADC | 0.3mm | 短而粗，远离干扰 |
| GPIO | 0.3mm | 普通信号 |
| PWM | 0.3mm | 远离 ADC |

**过孔：**
- 电源过孔：≥0.6mm
- 信号过孔：0.3mm
- 地过孔：0.4mm

### 4.5 铺铜规则

**顶层和底层：**
1. 全部铺 GND 铜
2. 电源网络单独铺铜
3. 天线区域禁止铺铜（ESP32 下方 5mm）

**设置方法：**
1. 点击"铺铜" → "添加铺铜"
2. 选择"顶层"或"底层"
3. 网络选择"GND"
4. 绘制铺铜区域
5. 确认

### 5.1 设置设计规则

- 信号线：0.2mm/0.2mm
- 电源线：0.5mm/0.3mm
- 过孔：0.4mm

**设置方法：**
1. 点击"设计" → "设计规则"

2. 等待检查完成

## 8. 焊接组装

### 8.1 焊接顺序

1. 电阻电容（矮的元件）
2. 三极管二极管
3. 排针接线端子
4. LED 蜂鸣器
5. 电源模块
6. ESP32-C3（最后）

### 8.2 焊接要点

**注意事项：**
- 电烙铁温度：350°C
- 焊接时间：<3 秒/焊点
- 先练习再焊接
- 检查虚焊和短路

**检查项目：**
- [ ] 所有焊点光亮
- [ ] 无虚焊
- [ ] 无短路
- [ ] 元件方向正确
- [ ] 极性元件方向正确

## 9. 测试验证

### 9.1 上电前检查

**万用表检查：**
1. 5V 对地电阻：>100Ω
2. 3.3V 对地电阻：>100Ω
3. 电池对地电阻：>1kΩ

### 9.2 上电测试

**步骤：**
1. 连接电池
2. 测量 5V 电压：4.8-5.2V
3. 测量 3.3V 电压：3.2-3.4V
4. 按复位键
5. 观察 LED 是否闪烁

### 9.3 功能测试

## 10. 常见问题

### Q1: 立创 EDA 打不开？
**A:** 使用 Chrome/Edge浏览器，清除缓存后重试。

### Q2: 元件库找不到元件？
**A:** 使用"万能库"搜索，或手动绘制封装。

### Q3: PCB 打样失败？
**A:** 检查 DRC，确保无错误和警告。

### Q4: 焊接后无法上电？
**A:** 检查电源极性，测量短路。

### Q5: ESP32 无法烧录？
**A:** 检查 GPIO0 是否接地（下载模式）。
