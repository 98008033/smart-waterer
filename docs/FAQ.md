## 硬件设计

### Q6: PCB 尺寸可以改吗？

**A:** 可以！推荐尺寸 80×60mm，但可以根据需求调整。

### Q7: 可以用其他 ESP32 型号吗？

**A:** 可以！ESP32/ESP32-S2/ESP32-S3 都可以，注意引脚可能不同。

### Q8: 土壤湿度传感器用哪种好？

**A:** 推荐电容式（耐腐蚀），不推荐电阻式（易腐蚀）。

### Q9: 水泵流量太小怎么办？

**A:** 
- 更换大流量水泵
- 延长浇水时间
- 优化水管路径

### Q10: 电池续航不够怎么办？

**A:**
- 增加电池容量
- 优化低功耗代码
- 减少上报频率
- 增加太阳能充电

## 固件开发

### Q11: 用什么开发环境？

**A:** 推荐 Arduino IDE，也可以用 ESP-IDF。

### Q12: 固件代码在哪里？

**A:** 参考 [固件开发指南](FIRMWARE_DEVELOPMENT.md)，有完整示例代码。

### Q13: 无法烧录固件怎么办？

**A:**
1. 检查 USB 线是否完好
2. 检查驱动是否安装
3. GPIO0 接地（进入下载模式）
4. 重启电脑重试

### Q14: 如何调试代码？

**A:**
- 使用串口监视器
- 添加 Serial.println()
- 使用 LED 指示状态
- 使用逻辑分析仪

### Q15: OTA 更新失败怎么办？

**A:**
1. 检查网络连接
2. 检查固件 URL
3. 检查 Flash 空间
4. 使用串口烧录

## 云平台

### Q16: 必须用腾讯云吗？

**A:** 不是！也可以用阿里云 IoT、OneNET、Blinker 等。

### Q17: 腾讯云免费额度够用吗？

**A:** 完全够用！个人项目每月 100 万消息，每天可以用 3 万多次。

### Q18: 如何获取设备三元组？

**A:**
1. 登录腾讯云 IoT 控制台
2. 进入产品 → 设备列表
3. 点击设备名称
4. 复制 ProductID、DeviceName、DeviceSecret

### Q19: MQTT 连接失败怎么办？

**A:**
1. 检查三元组是否正确
2. 检查网络是否通畅
3. 检查时间是否同步
4. 查看错误码

### Q20: 设备离线怎么办？

**A:**
1. 检查 WiFi 连接
2. 检查电源是否充足
3. 检查 MQTT 配置
4. 重启设备

## 故障排查

### Q31: 设备上电无反应

**排查步骤：**
1. 检查电池是否有电
2. 检查电源开关
3. 测量 3.3V/5V 电压
4. 检查是否有短路

### Q32: WiFi 连不上

**排查步骤：**
1. 检查 WiFi 密码
2. 检查信号强度
3. 检查路由器设置
4. 重启设备

### Q33: 传感器读数异常

**排查步骤：**
1. 检查接线是否正确
2. 检查传感器是否损坏
3. 校准传感器
4. 更换传感器

### Q34: 水泵不工作

**排查步骤：**
1. 检查水泵供电
2. 检查三极管是否导通
3. 检查水管是否堵塞
4. 更换水泵

### Q35: 小程序无法连接

**排查步骤：**
1. 检查网络
2. 检查 AppID 配置
3. 检查云开发环境
4. 重新登录
