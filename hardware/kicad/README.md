## 📥 导入方法

### 方法 1：直接打开（推荐）

1. 安装 KiCad 7.0+
   - 官网：https://www.kicad.org/download/
   - 选择对应系统版本下载

2. 打开项目
   ```
   启动 KiCad
   文件 → 打开项目
   选择：smart-waterer.kicad_pro
   ```

3. 查看电路图
   ```
   双击 smart-waterer.kicad_sch
   或右键 → 用 Eeschema 打开
   ```

4. 查看 PCB
   ```
   双击 smart-waterer.kicad_pcb
   或右键 → 用 Pcbnew 打开
   ```

### 方法 2：从立创 EDA 导入

如果已有立创 EDA 设计：

1. 立创 EDA 导出 KiCad 格式
   ```
   文件 → 导出 → KiCad
   选择原理图/PCB
   下载文件
   ```

2. KiCad 导入
   ```
   文件 → 导入
   选择立创 EDA 导出文件
   ```

## 🔧 使用指南

### 打开电路图后

**检查项目：**
- [ ] 所有元件已放置
- [ ] 网络连接正确
- [ ] 电源网络（5V/3.3V/GND）正确
- [ ] NTC 温度传感器电路正确
- [ ] TVS 浪涌保护电路正确
- [ ] 保险丝过流保护电路正确

**修改元件：**
```
双击元件 → 修改属性
可修改：
- 位号（Reference）
- 值（Value）
- 封装（Footprint）
- 立创编号（LCSC）
```

**添加连线：**
```
工具栏 → 放置导线（W）
点击起点 → 拖动到终点 → 点击确认
```

**添加网络标签：**
```
工具栏 → 放置网络标签（L）
放置到导线上 → 输入网络名
```

### 打开 PCB 后

**板框设置：**
```
文件 → 板框属性
修改尺寸：180×120mm
```

**元件布局：**
```
拖动元件到目标位置
右键 → 旋转（R）
右键 → 镜像（F）
```

**布线：**
```
工具栏 → 添加导线（X）
点击焊盘 → 拖动到目标焊盘
```

**铺铜：**
```
工具栏 → 添加填充区域（B）
选择网络：GND
设置 clearance：0.3mm
```

**DRC 检查：**
```
工具 → 设计规则检查
运行检查
修复所有错误
```

## 🎯 关键布局

### 元件布局要点

```
┌────────────────────────────────┐
│ USB-C → TVS → TP5100          │  ← 左侧（电源入口）
│                                │
│ 保护板 → 保险丝 → 稳压器       │  ← 中部（电源处理）
│                                │
│ ESP32-C3（居中）               │  ← 中央（主控）
│                                │
│ 传感器接口（顶部边缘）         │  ← 土壤/水位/NTC
│                                │
│ 执行器接口（右侧边缘）         │  ← 水泵/LED/蜂鸣器
└────────────────────────────────┘
```

### NTC 温度传感器

**重要：** NTC 不焊接在 PCB 上，需要飞线到电池表面！

```
PCB 上的 NTC 焊盘 ──→ 高温硅胶线 ──→ 电池表面
                          │
                         导热胶固定
```

**走线要求：**
- 使用双绞线
- 尽量短（<10cm）
- 远离 PWM 和大电流线
- 包地屏蔽

## 📤 导出生产文件

### Gerber 文件

**步骤：**
```
文件 → 导出 → Gerber (.gbr)
选择层：
☑ F.Cu（顶层走线）
☑ B.Cu（底层走线）
☑ F.SilkS（顶层丝印）
☑ B.SilkS（底层丝印）
☑ F.Mask（顶层阻焊）
☑ B.Mask（底层阻焊）
☑ Edge.Cuts（板框）
设置：
- 格式：2:5
- 单位：mm
- 钻孔文件：Excellon
生成 ZIP 文件
```

### 钻孔文件

**步骤：**
```
文件 → 导出 → 钻孔文件 (.drl)
格式：Excellon
单位：mm
零点：绝对
生成文件
```

### BOM 文件

**步骤：**
```
工具 → 生成物料清单
选择 CSV 格式
包含：
☑ 位号
☑ 值
☑ 封装
☑ LCSC 编号
生成文件
```

### 坐标文件

**步骤：**
```
文件 → 导出 → 位置文件 (.pos)
格式：CSV
单位：mm
包含：
☑ 位号
☑ X 坐标
☑ Y 坐标
☑ 旋转角度
☑ 面（Top/Bottom）
生成文件
```

## 🛠️ 常见问题

### Q1: KiCad 打不开文件？
**A:** 确保使用 KiCad 7.0+ 版本。旧版本可能不兼容。

### Q2: 元件封装找不到？
**A:** 
```
偏好设置 → 管理封装库
添加立创 KiCad 库：
https://github.com/uXnECu/kicad-libraries
```

### Q3: 如何添加立创编号？
**A:** 
```
双击元件 → 属性
添加字段：LCSC
输入立创编号（如 C2040789）
```

### Q4: 如何下单 PCB？
**A:** 
```
导出 Gerber 和钻孔文件
打包成 ZIP
上传到嘉立创：https://www.jlcpcb.com
```

### Q5: NTC 如何固定到电池？
**A:** 
1. 准备高温硅胶线（24AWG）
2. 焊接到 PCB 的 NTC 焊盘
3. NTC 贴片到电池表面
4. 使用导热胶固定
5. 热缩管绝缘保护

## ⚠️ 注意事项

1. **NTC 温度传感器** 必须贴电池表面，不要焊接在 PCB 上
2. **电源走线** 宽度≥1.0mm，防止过流发热
3. **ESP32 天线** 下方禁止铺铜（设置 keepout）
4. **TVS 二极管** 尽量靠近 USB 入口
5. **保险丝** 靠近电池正极
6. **三防漆** 喷涂前遮蔽连接器
7. **首次打样** 建议 5 片（便宜，可测试）
