## 📥 使用方法

### 1. 安装 KiCad 9.0+

**下载：** https://www.kicad.org/download/

选择你的系统版本：
- Windows：安装包
- macOS：DMG 或 Homebrew
- Linux：APT 或 Flatpak

### 2. 打开项目

**方法 A：直接打开项目文件**
```
启动 KiCad
文件 → 打开项目
选择：smart-waterer.kicad_pro
```

**方法 B：单独打开文件**
```
双击 smart-waterer.kicad_sch → 用 Eeschema 打开电路图
双击 smart-waterer.kicad_pcb → 用 Pcbnew 打开 PCB
```

### 3. 检查设计

**电路图检查：**
- [ ] 所有元件已放置
- [ ] 网络连接正确
- [ ] 电源网络（5V/3.3V/GND）正确
- [ ] NTC 温度传感器电路正确
- [ ] TVS 浪涌保护电路正确
- [ ] 保险丝过流保护电路正确

**PCB 检查：**
- [ ] 板框尺寸 180×120mm
- [ ] 所有元件已放置
- [ ] 电源走线宽度≥1.0mm
- [ ] 底层 GND 铺铜完整
- [ ] ESP32 天线区域 keepout 已设置
- [ ] DRC 检查通过

### 4. 导出生产文件

**Gerber 文件（PCB 制造）：**
```
文件 → 导出 → Gerber
选择层：
☑ F.Cu（顶层走线）
☑ B.Cu（底层走线）
☑ F.SilkS（顶层丝印）
☑ B.SilkS（底层丝印）
☑ F.Mask（顶层阻焊）
☑ B.Mask（底层阻焊）
☑ Edge.Cuts（板框）
生成 ZIP 文件
```

**钻孔文件：**
```
文件 → 导出 → 钻孔文件
格式：Excellon
生成文件
```

**BOM 文件（元件采购）：**
```
工具 → 生成物料清单
选择 CSV 格式
包含立创编号
生成文件
```

**坐标文件（SMT 贴片）：**
```
文件 → 导出 → 位置文件
格式：CSV
生成文件
```

## 🔧 修改指南

### 添加元件

**在 Eeschema 中：**
```
1. 右侧 → 放置符号
2. 搜索元件（如 R、C、Q）
3. 放置到画布
4. 双击修改属性
5. 添加立创编号（LCSC 字段）
```

### 修改连线

**在 Eeschema 中：**
```
1. 工具栏 → 放置导线（W）
2. 点击起点 → 拖动到终点
3. 添加网络标签（L）
```

### 修改 PCB 布局

**在 Pcbnew 中：**
```
1. 拖动元件到目标位置
2. 右键 → 旋转（R）
3. 工具栏 → 添加导线（X）
4. 工具栏 → 添加铺铜（B）
```

### 运行 DRC

**在 Pcbnew 中：**
```
工具 → 设计规则检查
运行检查
修复所有错误
```

## ⚠️ 注意事项

1. **NTC 温度传感器** 必须贴电池表面，不要焊接在 PCB 上
2. **电源走线** 宽度≥1.0mm，防止过流发热
3. **ESP32 天线** 下方禁止铺铜（设置 keepout）
4. **TVS 二极管** 尽量靠近 USB 入口
5. **保险丝** 靠近电池正极
6. **三防漆** 喷涂前遮蔽连接器
7. **首次打样** 建议 5 片（便宜，可测试）
