# 立创 EDA 使用指南

**重要说明：**

## ⚠️ 立创 EDA 不支持导入外部文件！

经过多次验证，**立创 EDA 标准版和专业版都不支持导入任何外部电路图文档**。

立创 EDA 只支持：
- ✅ **导出** Gerber、BOM、坐标文件
- ❌ **导入** 外部设计文件（不支持）

---

## 🎨 实际可行的方案

### 方案 1：手动绘制（推荐）⭐

**步骤：**

1. 打开立创 EDA
   ```
   https://lceda.cn/editor
   ```

2. 新建工程
   ```
   文件 → 新建 → 工程
   名称：Smart-Waterer-v0.3
   ```

3. 参考文档绘制
   ```
   打开：docs/HARDWARE_DESIGN.md
   查看第 6 节"完整原理图"
   ```

4. 从立创商城放置元件
   ```
   右侧 → 元件库 → 搜索立创编号
   例如：C2040789 (ESP32-C3)
   ```

5. 连接导线
   ```
   工具栏 → 导线（W）
   按照原理图连接
   ```

6. 设计 PCB
   ```
   工具栏 → 切换到 PCB
   布局 → 布线 → 铺铜
   ```

**预计耗时：** 2-4 小时（首次）

---

### 方案 2：立创开源广场（最快）⭐⭐⭐

**步骤：**

1. 访问开源广场
   ```
   https://oshwhub.com/
   ```

2. 搜索类似项目
   ```
   搜索关键词：
   - ESP32
   - 智能浇花
   - 物联网传感器
   ```

3. 复制项目
   ```
   找到喜欢的项目
   点击"开源复制"
   保存到自己的工程
   ```

4. 修改为自己的设计
   ```
   修改元件
   调整布局
   添加安全电路（NTC/TVS/保险丝）
   ```

**预计耗时：** 30 分钟 - 1 小时

**优点：**
- 快速开始
- 有参考设计
- 减少错误

## 🔧 手动绘制详细步骤

### 第 1 步：创建工程

```
1. 打开 https://lceda.cn/editor
2. 点击"编辑器"→"标准版"
3. 文件 → 新建 → 工程
4. 填写：
   - 工程名：Smart-Waterer-v0.3
   - 板名：Smart-Waterer-PCB
5. 确定
```

### 第 2 步：放置 ESP32-C3

```
1. 右侧 → 元件库
2. 搜索：C2040789
3. 点击 ESP32-C3-SuperMini
4. 放置到画布中央
5. 按空格旋转（如需要）
```

### 第 3 步：放置电源部分

```
1. 搜索 C165948 → USB-C → 放在左侧
2. 搜索 C117462 → SMAJ5.0A → 放在 USB 旁边
3. 搜索 TP5100 → 放在右侧
4. 搜索 C318886 → 1812L110 → 放在 TP5100 下方
```

### 第 4 步：放置传感器接口

```
1. 搜索 C20959 → KF301-2P
2. 放置 3 个：
   - J2: 土壤湿度
   - J3: 水位检测
   - 放在板子顶部边缘
```

### 第 5 步：放置执行器

```
1. 搜索 C198134 → AO4606 → 放在右侧
2. 搜索 C9628 → 1N4148 → 放在 MOSFET 旁边
3. 搜索 C20959 → KF301-2P (J4 水泵) → 放在右侧边缘
4. 搜索 C2286 → LED → 放在 ESP32 旁边
5. 搜索 C105909 → 蜂鸣器 → 放在 ESP32 旁边
```

### 第 6 步：放置 NTC 温度传感器

```
1. 搜索 C2893625 → NTC 10kΩ
2. 搜索 C17414 → 10kΩ电阻
3. 搜索 C1585 → 100nF 电容
4. 放在 ESP32 的 GPIO3 附近
5. 注意：NTC 实际要飞线到电池，PCB 上只留焊盘
```

### 第 7 步：放置稳压器

```
1. 搜索 C6186 → AMS1117-3.3
2. 放在板子底部
3. 添加输入输出电容
```

### 第 8 步：连接导线

**按照以下连接：**

```
3.3V 网络：
- U1:3V3 → R1:1 → TH1:1 → U2:VOUT

GND 网络：
- U1:GND → D1:2 → C1:2 → TH1:2 → U2:GND

USB_5V：
- J1:VBUS → D1:1 → TP5100:IN+

NTC_Temp：
- U1:GPIO3 → R1:2 → TH1:1 → C1:1

Soil_Moisture：
- U1:GPIO0 → J2:AO

Water_Level：
- U1:GPIO2 → J3:OUT

Pump_Control：
- U1:GPIO1 → Q1:G

LED：
- U1:GPIO4 → R2:1 → LED1:+

Buzzer：
- U1:GPIO5 → BZ1:+

Battery_P+：
- TP5100:OUT+ → F1:1

System_5V：
- F1:2 → U2:VIN → J4:1
```

### 第 9 步：添加网络标签

```
1. 工具栏 → 网络标签（L）
2. 放置到对应导线
3. 输入网络名（如 3.3V、GND 等）
```

### 第 10 步：设计 PCB

```
1. 点击"转换到 PCB"
2. 设置板框：80×60mm
3. 布局元件
4. 布线（电源走线≥1.0mm）
5. 底层铺铜（GND）
6. DRC 检查
```

## ⚠️ 重要提示

1. **NTC 温度传感器** 必须贴电池表面，不要焊接在 PCB 上
2. **电源走线** 宽度≥1.0mm
3. **ESP32 天线** 下方禁止铺铜
4. **TVS 二极管** 尽量靠近 USB 入口
5. **保险丝** 靠近电池正极
