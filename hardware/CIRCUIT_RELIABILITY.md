## 3. 水泵驱动可靠性

### 3.1 驱动电路

**电路设计：**
```
GPIO1 ──→ 1kΩ ──→ S8050 基极
S8050 发射极 ──→ GND
S8050 集电极 ──→ 水泵负极
水泵正极 ──→ 5V
1N4148 反向并联 ──→ 水泵两端
```

### 3.2 可靠性分析

| 项目 | 计算值 | 额定值 | 安全裕度 | 结论 |
|------|--------|--------|---------|------|
| 基极电流 | 2.6mA | 50mA | 5% | ✅ 充足 |
| 集电极电流 | 300mA | 500mA | 60% | ✅ 充足 |
| 功耗 | 0.18W | 0.625W | 29% | ✅ 充足 |
| 温升 | <20°C | 150°C | 13% | ✅ 充足 |

**计算过程：**
```
基极电流：Ib = (3.3V - 0.7V) / 1kΩ = 2.6mA
集电极电流：Ic = 300mA (水泵)
放大倍数：β = Ic / Ib = 115 (S8050 β>100)
功耗：P = Vce × Ic = 0.6V × 0.3A = 0.18W
```

### 3.3 保护措施

| 保护类型 | 实现方式 | 效果 |
|---------|---------|------|
| 续流保护 | 1N4148 二极管 | ✅ 防止反电动势 |
| 限流保护 | 基极 1kΩ电阻 | ✅ 限制基极电流 |
| 过温保护 | 自然散热 | ✅ 温升<20°C |
| 短路保护 | 无 | ❌ 建议添加 |

**结论：** 驱动电路设计合理，建议添加保险丝。
