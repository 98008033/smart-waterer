# 🔧 PCB 安全整改修改指南

**版本：** v1.0  
**日期：** 2026-03-13  
**适用：** 智能浇花机 PCB v0.1  
**优先级：** ⚠️ **P0 - 必须修改**

---

## 📋 修改概述

需要在原 PCB 基础上添加以下安全电路：

1. **NTC 温度传感器电路** - 电池温度监控
2. **自恢复保险丝** - 过流保护
3. **TVS 二极管** - 浪涌保护
4. **电池保护板接口** - 2 串锂电保护

---

## 1️⃣ NTC 温度传感器电路

### 电路原理图

```
         3.3V
          │
         ┌┴┐
         │ │ 10kΩ 上拉电阻 (R_NTC_PULLUP)
         └┬┘
          │
          ├───┬─── 连接到 ESP32 GPIO3 (ADC1_CH3)
          │   │
         ┌┴┐ ┌┴┐ 100nF 滤波电容 (C_NTC_FILT)
         │ │ │ │
         └┬┘ └┬┘
          │   │
          │  GND
          │
         ┌┴┐
         │ │ NTC 热敏电阻 10kΩ B3950 (TH1)
         └┬┘
          │
         GND
```

### PCB 布局要求

**NTC 安装位置：**
```
电池仓
┌─────────────────┐
│  18650  │ 18650  │
│  电池 1  │ 电池 2  │
│         ●       │  ← NTC 贴在此处（两节电池中间）
│        TH1      │     使用导热胶固定
└─────────────────┘
```

**走线要求：**
- NTC 到 ESP32 的走线尽量短
- 远离 PWM 线和大电流线
- 使用包地保护（两侧走 GND）

### 元件封装

| 元件 | 封装 | 数量 |
|------|------|------|
| NTC 热敏电阻 | 0805 或 引线式 | 1 |
| 上拉电阻 | 0805 | 1 |
| 滤波电容 | 0805 | 1 |
| 2Pin 端子 | KF301-2P | 1 |

## 2️⃣ 自恢复保险丝电路

### 电路原理图

```
电池+ (B+)
   │
  ┌┴┐
  │ │ 100kΩ 放电电阻 (R_DISCHARGE)
  └┬┘
   │
  GND

电池+ (B+)
   │
  ═══ 1A 自恢复保险丝 (F1 1812L110)
   │
   ├───→ TP5100 B+
   │
   ├───→ 电池电压检测分压电阻
```

### PCB 布局要求

**保险丝位置：**
```
电池输入 ──→ [保险丝 F1] ──→ 电源输入
                 │
                靠近电池入口
```

**走线要求：**
- 保险丝前后走线加粗（≥1.0mm）
- 远离发热元件
- 便于更换（如使用插座式）

### 元件封装

| 元件 | 封装 | 数量 |
|------|------|------|
| 自恢复保险丝 | 1812 | 1 |
| 放电电阻 | 0805 | 1 |

## 3️⃣ TVS 浪涌保护电路

### 电路原理图

```
USB-C 5V
   │
  ═══ TVS 二极管 (D_TVS SMAJ5.0A)
   │
  GND

   │
   ├───→ TP5100 IN+
```

### PCB 布局要求

**TVS 位置：**
```
USB-C 接口 ──→ [TVS D1] ──→ TP5100
                  │
                 GND
              尽量靠近 USB 口
```

**走线要求：**
- TVS 到 GND 的走线短而粗
- TVS 尽量靠近 USB 入口
- 5V 主电源线加粗（≥1.0mm）

### 元件封装

| 元件 | 封装 | 数量 |
|------|------|------|
| TVS 二极管 | DO-214AC (SMA) | 1 |

## 4️⃣ 电池保护板接口

### 电路原理图

```
18650 电池组 (2 串)
      │
      ├── B- ──→ 保护板 B-
      │
      ├── P- ──→ 保护板 P-
      │
保护板 ──┬── P+ ──→ 保险丝 F1 ──→ 系统电源
         │
         └── T ──→ NTC ──→ GPIO3

保护板参数：
- B-: 电池负极
- P-: 放电/充电负极
- P+: 输出正极
- T:  温度检测（NTC）
```

### PCB 布局要求

**保护板位置：**
```
┌───────────────────────┐
│   18650 电池座        │
│   ┌───┐ ┌───┐        │
│   │ 1 │ │ 2 │        │
│   └───┘ └───┘        │
│      │     │          │
│      └──┬──┘          │
│         │             │
│   ┌─────┴─────┐       │
│   │ 保护板    │       │  ← 放在电池座旁边
│   │ 2 串带温度 │       │
│   └─────┬─────┘       │
│         │             │
└─────────┼─────────────┘
          │
         GND
```

**接线要求：**
- 电池线使用高温硅胶线（24AWG）
- B+/B- 走线加粗（≥1.5mm）
- NTC 线使用双绞线（减少干扰）

## 6️⃣ PCB 修改步骤

### 方法 A：飞线修改（快速验证）

**适用：** 已有 PCB，快速验证安全电路

**步骤：**
1. **NTC 电路飞线：**
   - 3.3V → 10kΩ电阻 → GPIO3
   - GPIO3 → NTC → GND
   - NTC 用导热胶贴在电池表面

2. **保险丝飞线：**
   - 剪断电池+ 输入线
   - 串联 1A 自恢复保险丝

3. **TVS 飞线：**
   - USB 5V → TVS → GND
   - TVS 靠近 USB 口焊接

**优点：** 快速、成本低  
**缺点：** 不美观、可靠性稍差

---

### 方法 B：重新打样（推荐）

**适用：** 正式量产

**步骤：**
1. 在 LCEDA 中打开原工程
2. 按上述电路图添加元件
3. 重新布局布线
4. 下单打样（嘉立创 5 片¥15）

**修改要点：**
- 添加 NTC 电路（3 个元件）
- 添加保险丝（1 个元件）
- 添加 TVS（1 个元件）
- 预留保护板安装位

**优点：** 美观、可靠、便于生产  
**缺点：** 需要等待 3-5 天

## 🔟 常见问题

### Q1: NTC 读数不准？
**A:** 检查上拉电阻是否为 10kΩ，ADC 参考电压是否正确。

### Q2: 保险丝频繁跳闸？
**A:** 检查负载电流是否超过 1A，或更换 1.5A 规格。

### Q3: TVS 发烫？
**A:** 正常工作时不应发烫，检查是否持续浪涌。
