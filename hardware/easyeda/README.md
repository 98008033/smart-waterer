## 📐 手动绘制指南（推荐）

### 步骤 1：创建新项目

1. 打开 https://lceda.cn/editor
2. 点击 **编辑器** → **标准版**
3. **文件** → **新建** → **工程**
4. 填写：
   - 工程名：`Smart-Waterer-v0.3`
   - 板名：`Smart-Waterer-PCB`
5. 点击确定

### 步骤 2：绘制电路图

**参考文档：** `docs/HARDWARE_DESIGN.md` 第 6 节"完整原理图"

**元件清单：**

| 位号 | 元件 | 立创编号 | 封装 |
|------|------|---------|------|
| U1 | ESP32-C3-SuperMini | C2040789 | ESP32-C3-SuperMini |
| TH1 | NTC 10kΩ B3950 | C2893625 | R0805 |
| R1 | 10kΩ | C17414 | R0805 |
| C1 | 100nF | C1585 | C0805 |
| D1 | SMAJ5.0A TVS | C117462 | DO-214AC |
| U2 | TP5100 | - | TP5100_Module |
| F1 | 1812L110 保险丝 | C318886 | 1812 |
| U3 | MP1584 | - | MP1584_Module |

| J1 | USB-C | C165948 | USB-C-16Pin |
| Q1 | AO4606 | C198134 | SOIC-8 |

| LED1 | LED 3mm | C2286 | LED-3mm |
| BZ1 | 蜂鸣器 | C105909 | Buzzer-3.3V |

**绘制步骤：**

1. **放置元件**
   ```
   右侧 → 元件库 → 搜索立创编号
   点击元件 → 放置到画布
   ```

2. **连接导线**
   ```
   工具栏 → 导线（W）
   点击引脚 → 拖动到目标引脚
   ```

3. **添加网络标签**
   ```
   工具栏 → 网络标签（N）
   放置到导线上 → 输入网络名
   ```

**关键网络：**
- `USB_5V` - USB 5V 输入
- `GND` - 地线
- `3.3V` - 3.3V 电源
- `NTC_Temp` - NTC 温度信号
- `Battery_P+` - 电池正极
- `System_5V` - 系统 5V
- `Pump_Control` - 水泵控制
- `Soil_Moisture` - 土壤湿度 ADC
- `Water_Level` - 水位检测
- `LED` - LED 控制
- `Buzzer` - 蜂鸣器控制

### 步骤 3：设计 PCB

**板框设置：**
1. **机械层 → 板框**
2. 绘制矩形：80×60mm
3. 4 角添加 M3 安装孔（孔径 3.2mm，距边 3mm）

**元件布局：**

```
┌────────────────────────────────┐
│ USB-C → TVS → TP5100          │  ← 左侧（电源入口）
│                                │
│ 保护板 → 保险丝 → 稳压器       │  ← 中部（电源处理）
│                                │
│ ESP32-C3（居中）               │  ← 中央（主控）
│                                │
│ 传感器接口（顶部边缘）         │  ← 土壤/水位/NTC
│                                │
│ 执行器接口（右侧边缘）         │  ← 水泵/LED/蜂鸣器
└────────────────────────────────┘
```

**布线规则：**

| 网络 | 线宽 | 说明 |
|------|------|------|
| 5V 电源 | ≥1.0mm | 水泵/传感器供电 |
| 3.3V 电源 | ≥0.5mm | ESP32 供电 |
| GND | ≥1.0mm | 底层完整铺铜 |
| 信号线 | 0.3mm | GPIO/ADC |
| NTC 走线 | 0.3mm | 包地保护 |

**铺铜设置：**
1. **底层 → 铺铜**
2. 网络选择：`GND`
3.  clearance：0.3mm
4.  勾选"移除死铜"

**Keepout 设置：**
- ESP32 天线区域（板载天线下方）禁止铺铜
- 尺寸：10×8mm
- 位置：ESP32-C3 模块上方

### 步骤 4：DRC 检查

```
工具 → DRC 检查
运行检查
修复所有错误
```

## 🔧 NTC 温度传感器安装

**重要：** NTC 不焊接在 PCB 上，需要飞线到电池表面！

**步骤：**
1. PCB 上预留 2Pin 焊盘（NTC 接口）
2. 准备高温硅胶线（24AWG，2 芯）
3. 焊接到 PCB 焊盘
4. NTC 贴片到电池表面（两节电池中间）
5. 使用导热胶固定
6. 热缩管绝缘保护

**走线要求：**
- 使用双绞线
- 长度<10cm
- 远离 PWM 和大电流线
- 包地屏蔽

## 📤 导出生产文件

### Gerber 文件
```
文件 → 导出 → Gerber
选择所有层：
- TopLayer（顶层走线）
- BottomLayer（底层走线）
- TopPaste（顶层焊盘）
- TopSilk（顶层丝印）
- TopSolder（顶层阻焊）
- BottomSolder（底层阻焊）

```

### BOM 文件
```
文件 → 导出 → BOM
选择 CSV 格式
用于立创商城配单
```

### 坐标文件
```
文件 → 导出 → Pick&Place
用于 SMT 贴片（可选）
```

## ⚠️ 注意事项

1. **NTC 温度传感器** 必须贴电池表面，不要焊接在 PCB 上
2. **电源走线** 宽度≥1.0mm，防止过流发热
3. **ESP32 天线** 下方禁止铺铜
4. **TVS 二极管** 尽量靠近 USB 入口
5. **保险丝** 靠近电池正极
6. **三防漆** 喷涂前遮蔽连接器
7. **首次打样** 建议 5 片（便宜，可测试）
