## 📥 导入方法

### 方法 1：直接导入（推荐）

1. 打开嘉立创 EDA 标准版
   - 网址：https://lceda.cn/editor

2. 导入电路图
   ```
   文件 → 导入 → 文件
   选择：smart-waterer-schematic.json
   确认导入
   ```

3. 导入 PCB
   ```
   文件 → 导入 → 文件
   选择：smart-waterer-pcb.json
   确认导入
   ```

### 方法 2：复制粘贴

如果 JSON 导入不成功，可以：

1. 打开 `HARDWARE_DESIGN.md` 文档
2. 查看第 6 节"完整原理图"
3. 根据 ASCII 原理图手动绘制

### 方法 3：使用立创 EDA 专业版

专业版导入步骤：
```
文件 → 导入 → Altium Designer
选择 JSON 文件
确认转换
```

## 🎯 关键布局

### 元件布局要点

```
┌────────────────────────────────┐
│ USB-C → TVS → TP5100          │  ← 电源入口（左侧）
│                                │
│ 保护板 → 保险丝 → 稳压器       │  ← 电源处理（中部）
│                                │
│ ESP32-C3（居中）               │  ← 主控（远离干扰）
│                                │
│ 传感器接口（顶部）             │  ← 土壤/水位/NTC
│                                │
│ 执行器接口（右侧）             │  ← 水泵/LED/蜂鸣器
└────────────────────────────────┘
```

### NTC 温度传感器

**重要：** NTC 不焊接在 PCB 上，需要飞线到电池表面！

```
PCB 上的 NTC 焊盘 ──→ 高温硅胶线 ──→ 电池表面
                          │
                         导热胶固定
```

**走线要求：**
- 使用双绞线
- 尽量短（<10cm）
- 远离 PWM 和大电流线

## 🛠️ 常见问题

### Q1: JSON 导入失败？
**A:** 嘉立创 EDA 标准版支持 JSON 导入，如失败可尝试：
- 使用专业版导入
- 或手动绘制（参考 HARDWARE_DESIGN.md）

### Q2: 元件封装找不到？
**A:** 
- ESP32-C3-SuperMini：在立创商城搜索"C2040789"
- TP5100 模块：搜索"TP5100_Module"
- 其他元件使用标准封装

### Q3: 如何下单 PCB？
**A:** 
```
嘉立创 EDA → 制造 → PCB 下单
选择参数：
- 尺寸：80×60mm
- 层数：2 层
- 板厚：1.6mm
- 铜厚：1oz
- 阻焊：绿色
- 数量：5 片
价格：¥15（新人首单免费）
```

### Q4: NTC 如何固定到电池？
**A:** 
1. 准备高温硅胶线（24AWG）
2. 焊接到 PCB 的 NTC 焊盘
3. NTC 贴片到电池表面
4. 使用导热胶固定
5. 热缩管绝缘保护

## ⚠️ 注意事项

1. **NTC 温度传感器** 必须贴电池表面，不要焊接在 PCB 上
2. **电源走线** 宽度≥1.0mm，防止过流发热
3. **ESP32 天线** 下方禁止铺铜（已设置 keepout）
4. **TVS 二极管** 尽量靠近 USB 入口
5. **保险丝** 靠近电池正极
6. **三防漆** 喷涂前遮蔽连接器
